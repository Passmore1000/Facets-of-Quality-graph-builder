# Contributing

Thanks for your interest in improving this project.

## Ways to contribute

- Report bugs
- Suggest UX improvements
- Improve docs
- Open small focused pull requests

## Development setup

```bash
npm install
npm run dev
```

## Pull request guidelines

- Keep PRs focused and small
- Include a clear description of why the change is needed
- Run `npm run lint` and `npm run build` before opening a PR
- Update docs if behavior or setup changes

## Scope and maintenance

This is a community-friendly project, but not actively maintained as a product.
PRs may be reviewed as time allows.
Forks are always welcome.
